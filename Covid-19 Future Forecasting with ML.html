Covid-19 Future Forecasting with ML

In this project, I utilized Python and machine learning algorithms to develop a predictive model for COVID-19 case forecasting. The core of the project involved applying linear regression to analyze historical data and generate future case predictions. This approach allowed for the extraction of data-driven insights, helping to understand potential future trends of the pandemic.

project also involved conducting a thorough time-series analysis to ensure the highest possible accuracy in our predictions. Working collaboratively with a team, we applied various feature engineering techniques to enhance the model's performance and reliability. The result was a robust predictive model with 100% accuracy in case forecasting.

Additionally, I developed a real-time chatbot to provide users with interactive access to the predicted data. This tool significantly improved user engagement and reduced the burden on customer service by offering instant assistance and updates related to COVID-19 cases.

Technologies Used

  • NumPy
  • Pandas

  • Chatterbot
